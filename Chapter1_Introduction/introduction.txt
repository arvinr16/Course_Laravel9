==> Chapter 1 Introduction <==
=> Apa sebenarnya itu Laravel? = Laravel adalah Framework pembuat website yang sangat populer di Indonesia. Laravel ini berbasis bahasa-
                              Pemrograman PHP. Banyak dari perusahaan di Indonesia mengadopsi Laravel sebagai bagian dari sistem website-
                              perusahaan tersebut. Laravel juga memiliki banyak fitur bawaan, sehingga kita sebagai developer tidak-
                              perlu memikirkan hal-hal yang kecil atau sepele.
<========================================================================================================================================>
=> Sebelum menggunakan Laravel = Ada beberapa hal yang harus dipersiapkan terlebih dahulu sebelum menggunakan framework Laravel.
                              Kita bisa menggunakan:
                              1. XAMPP = Software yang menyediakan beberapa service, yang nantinya kita akan menggunakan service Apache-
                                         dan MySQL.
                              2. MAMP  = Gunakan software MAMPP bagi yang menggunakan device Apple atau laptop Mac sebagai alat untuk-
                                         menjalankan service Apache dan MySQL.
                              3. Composer = Dependency manager untuk PHP yang nantinya akan digunakan untuk mengelola library/ package-
                                            PHP secara otomatis, mirip seperti NPM di Javascript, PIP di Python. Fungsi utamanya adalah-
                                            untuk mengunduh libary PHP seperti Laravel, Symfony, Guzzle, dll, terus bisa mengatur versi-
                                            libary agar tidak bentrok, dll kegunaannya.
                              4. Visual Studio Code = Text editor atau kode editor yang nantinya bisa kita gunakan untuk mengedit,-
                                                      mengelola, dan mengembangkan kode program kita.


<========================================================================================================================================>
=> Outcomes: 1. Object Oriented Programming in PHP = Nantinya kita akan belajar paradigma OOP di dalam PHP.
             2. Basic of Relational Database = Nantinya kita akan mempelajari basic dari Relational Database, juga nantinya kita akan-
                                               belajar menggunakan/mengoperasikan MySQL.
             3. Migrations = Nantinya kita akan mempelajari Migrasi dan kegunaannya.
             4. Routing    = Nantinya kita akan mempelajari Routing pada website.
             5. Models, Views, Controllers = Nantinya kita akan mempelajari dasar dari Laravel yaitu Models, Views, dan Controllers.
             6. Eloquent ORM = Nantinya kita akan mempelajari dan mengimplementasikan Eloquent ORM(Object Relational Mapping).
             7. Authentication = Kemudian kita akan mengimplementasikan fitur Authentication, misalkan Login/Logout, forgot password,-
                                 register.
             8. Middleware    = Nantinya juga kita akan mempelajari apa itu Middleware.
             9. File Storage  = Kemudian nantinya kita juga akan memgimplementasikan File Storage pada website kita.
             10. Localization = Kita juga nantinya akan mengimplementasikan Localization pada website kita, misal kita bisa berganti bahasa-
                                Indonesia dan juga bahasa Inggris.
             11. Testing = Dan untuk yang terakhir kita akan melakukan Testing pada website Laravel yang sudah kita buat.
